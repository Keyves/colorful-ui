# Colorful-UI

基于Vue.js的UI组件库

## 特性

- 借鉴并融合了 Material Design 和 Ant Design 的设计理念
- 采用ECMAscript新特性构建
- 简单且丰富的接口设计

## 要求

- Vue.js(^2.1.0)

## 安装

```js
npm install --save colorful-ui
```

## 使用

```js
import ColorfulUI from 'colorful-ui'
import 'colorful-ui/dist/colorful-ui.min.css'

Vue.use(ColorfulUI)
```

## 兼容

支持现代浏览器和IE9及以上
