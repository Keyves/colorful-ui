# 更新日志

#### 版本号修订规则

- 主版号：不向下兼容的 API 修改
- 次版号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正
> 举例：版本号 1.2.3 （1为主版号，2为次版号，3为修订号）
